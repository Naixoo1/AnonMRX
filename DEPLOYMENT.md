# UMKM Karawang - Deployment Guide

## GitHub Pages Setup (Recommended)

### Step 1: Create GitHub Repository
1. Go to [github.com](https://github.com)
2. Create new repository: `umkm-karawang`
3. Upload all your UMKM files

### Step 2: Enable GitHub Pages
1. Go to repository Settings
2. Scroll to "Pages" section
3. Source: Deploy from a branch
4. Branch: main / (root)
5. Save

### Step 3: Access Your Site
- Your site will be available at: `https://[username].github.io/umkm-karawang/`

## Custom Domain Setup

### Option A: Free Subdomain
- Use GitHub Pages default URL
- Example: `username.github.io/umkm-karawang`

### Option B: Purchase Custom Domain ($10-15/year)
1. Buy domain from Namecheap or GoDaddy
2. Add domain in GitHub Pages settings
3. Configure DNS records

## Alternative: Netlify (Easier Setup)

### Quick Deploy
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop your UMKM folder
3. Get instant URL: `random-name.netlify.app`
4. Add custom domain in settings

## Custom Domain Recommendations

For UMKM Karawang website:
- `umkm-karawang.com` - Professional and clear
- `karawang-umkm.id` - Indonesian domain extension
- `ukmkarawang.com` - Short and memorable
